# trello-scrum-to-latex

Parses the Trello JSON data and writes it in LaTeX style format

## Installation

```bash
npm i -g trello-scrum-to-latex
```

## Trello Setup

Create a new list

## Usage

```bash
trello-scrum-to-latex --id [YOUR BOARD ID] --appkey [TRELLO APPKEY] --secret [TRELLO SECRET]
```
